### 3.1.1 golang 的特点
- 安全性高
- 开发简单
- 执行效率高
- 占用内存少

### 3.1.2 关于数据库
- mysql 是关系型数据库
    - 存取稳定安全可靠
    - 免费
    - 大量商业软件都在使用
- redis 是非关系型数据库
    - 存取速度非常快
    - key/value 形式存取
    - 大量用于缓存
