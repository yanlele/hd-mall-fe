# 产品文档

## 总体构成

### 公共部分
- 品牌
    - logo
    - 应用主题颜色
    - [色阶](https://ant.design/docs/spec/colors-cn)
    - 品牌推广： 一定要在最底下、以及少量信息在最上面
        - 最上面是提供的最重要的服务
        - 下面提供的其他服务
        - 说明
        - 版权
        - 申明等
- 公用header
    - 最重要的品类（一般在header部分）
    - 搜索入口
    - 重要的活动入口

- 搜索
    - 输入框搜索（一般在header搜索）
    - 筛选项（可以自定义）
    - 排序搜索搜索
        - 综合
        - 销量
        - 价格
        - 评价（也可以叫评分）
        
- 具体商品卡片
    - 产品展示图
    - 价格
    - 打标
    - 标题
    - 描述
    - 品牌
    - 发货地址
    - 店家联系方式
- 订单列表
	- 动态 table header
    	- 日期
        - 订单号
        - 商家联系方式
        - 品牌
        - 删除
    - 商品卡信息
    - 购买数量
    - 价格
    - 订单详情（跳转页面）
    - 物流详情
    - 操作
    	- 付款
        - 收货
        - 评价
    - 订单状态（待付款、已付款、待发货、已发货、已确认收获）

- 地址修改模态框
    - 地址信息（省/市）
    - 详细地址（区/县/镇/街/门牌号/具体楼层）
    - 收货人姓名
    - 收货人电话


### 首页
- 提供入口
    - **品类查找搜索**非常重要
        - 类别入口
        - 具体商品型号入口
        - 搜索入口（也在header）
    - 推广广告
        - 最重要的品类（一般在header部分）
        - 录播图（新品、爆款、品牌）
        - 活动
        - 推广
    - 分类卖爆款（品牌推荐）

### 搜索列表页面/类别入口落地/型号入口落地
- 筛选项（可以自定义）
    - 品牌
- 排序规则
    - 综合
    - 销量
    - 好评度
    - 价格区间
- 商品卡片列表
- 支持分页

### 商品详情页面
- 购买部分
    - 商品图介绍
        - 轮播图
        - 点击放大图
        - 相册图
    - 商品搜藏
    - 商品加入购物车
    - 商品标题
    - 商品价格（包含打折情况）
    - 附加信息：配送、商品情况、发货情况、销量、评价数量、好评度、商品保障信息、顾客权益、承诺服务等
    - 商品属性选择：颜色、品类、型号等选择（可以自定义）
    - 购买数量
- 商品预览：几乎都是静态贴图， 到时候后台系统配置即可
- 商品详情参数：后台配置几个， 展示方式以左右两栏展示
- 评论部分
    - 头像
    - 名字
    - 评价内容
    - 评价日期
    - 打分
### 订单页面
- 地址管理部分
    - 选择地址
    - 是否设置为默认地址
    - 地址管理（单独的页面， 跳转到另外的页面）
    - 地址修改模态框（单独的模态框 - 见公共部分）
       
- 确认订单信息：要用表格形式， 因为可能存在多种商品场景
    - 卖家信息
    - 商品信息描述
    - 用户选择的商品属性信息
    - 价格
    - 数量
    - 总计价格
    - 提交订单
    
### 支付页面（done）个体户没有办法做支付
